# Athena App

## Getting Started

### Prerequisites

- NPM
- Parcel bundler

### Installation

1. Clone the repository

   ```bash
   git clone [repo]
   ```

2. Navigate to the project directory

   ```bash
   cd client
   ```

3. Install dependencies

   ```bash
   npm install
   ```

4. Run the application

   ```bash
   npm start
   ```

5. Build the application

   ```bash
   npm run build
   ```
